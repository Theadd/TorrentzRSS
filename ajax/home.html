About TorrentzRSS!

TorrentzRSS is an open source project that processes torrentz.eu RSS results from your search queries and provides a set of advanced rules to merge multiple queries, sort them and exclude unwanted torrents from your search results in several ways.

Providing you with a fully customized RSS feed so you can add it to your favourite torrent application which will automatically download all torrents and periodically check for new additions to the results of your customized RSS feed.

Usage:

  • Type any search terms in the search box on the top.
  • In the sidebar, customize your search.
  • Finally, click Apply query & rules! or the button in the search box.

Example (Step by step)

Let's say we have the following search query to get the last episodes from our favorite TV shows:

"da vincis demons" | "falling skies" | continuum | defiance | "doctor who" | vikings | revolution | fringe | "game of thrones" | "marvel agents" | "orphan black" size > 500m size < 2g

As you can see, we've already specified two conditions to retrieve only torrents of size between 500 and 2000 MB. Try it out by pasting this search in the search box on top of the page and pressing enter or clicking 'Apply query & rules!' button.

You'll probably see lots of results, duplicated ones, results with a poor health (only 1 seeder), etc. But you only want the last episode from each TV show, with good quality and health. Let's do it, follow these steps:

  • First of all, set 'Order by' to 'Date' so we'll only get newest ones.
  • Add a new 'Handle duplicates' rule for TV shows and click 'Apply query & rules!' button to see what changed.

    You'll probably see few results since this rule only took the last episode from each TV show choosing the best match, which can be customized if you click on the 'Handle duplicates (TV)' rule you just created.

    It may not be the case but sometimes, there is an episode in 1080p quality but with only 1 seeder and lots of leechers, let's avoid this.

  • Add a new 'Evaluate condition' rule. Click on the new rule you've just created to open its parameters. Type "seeds > 100" without quotes to exclude results not matching this condition.
  • Drag & drop the 'Evaluate condition' rule before 'Handle duplicates (TV)' to exclude the ones with fewer seeds before removing all duplicates.

    Click again 'Apply query & rules!' to update your search.

Finally, we got what we were looking for, just copy & paste the 'RSS Feed' link into your favourite torrent application to automatically download everything periodically!

Advanced Search Parameters

Search parameters from torrentz.eu can also be used in the search box, those parameters are:

  • Mozilla Firefox - search for Mozilla and Firefox anywhere in the title
  • "Mozilla Firefox" - search exactly for "Mozilla Firefox"
  • Mozilla | Linux - search for Mozilla or Linux
  • Mozilla Firefox -beta - search for Mozilla AND Firefox but NOT beta
  • ^Mozilla - search for titles starting with Mozilla
  • movie* - match movie, movies or any other prefix - This is enabled by default on most queries
  • file: Fedora iso - deep search inside torrent for file named Fedora iso
  • file: Fedora size:3520802816 - deep search inside torrent for file named Fedora iso with size of 3520802816 bytes
  • Linux added < 7d or Linux added: 7d - search for Linux added less then 7 days ago
  • Linux added > 5y - search for Linux added more then 5 years ago. Available date units: hours, days, months, years
  • Linux size > 700m - search for Linux torrents with size bigger then 700megabytes. Available size units: kilo, mega, giga, tera
  • Linux size 700m - 1g - search for Linux torrents in size range of 700m and 1g
  • Linux seed > 100 - search for Linux torrents with more than 100 seeders. Available seed(s,ers), leech(ers,es) and total peer(s)
  • site:archive.org - search for torrents on archive.org only
